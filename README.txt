
In order to run the MentorGui.java program, click the run file. The authentication
window will pop-up and enter the username(admin) and password(a123) to access the
Mentors info panel. In this form user will enter the details and save the record 
to database. The form also contains the Reset button, which will be used to clear
the entered data. Upon clicking the save button, the data will be stored in the 
database.
